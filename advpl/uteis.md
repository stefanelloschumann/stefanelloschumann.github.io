## Formatação de datas

* **DTOC(\<data\>)**: Converte de data para caracter
* **DTOS(\<data>\)**: Converte de data para string no formato aaaammdd
* **STOD(\<aaaammdd\>)**: Converte de String para data no formato dd/mm/aaaa
* **Date()**: Retorna a data atual
